# ri-optimization

## Pre-requisites
- [Rust](https://www.rust-lang.org/)
- [Git](https://git-scm.com/downloads)

## Setup
Clone the repository:
```bash
git clone https://github.com/hiibolt/ri_optimization.git
```

Change into the directory of the project:
```bash
cd ri_optimization
```

Start the project:
```bash
cargo run
```
